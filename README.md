# 🐶 Про проєкт

Pets Founder створений для спрощення процесу адопції тварин та популяризації
відповідального ставлення до домашніх улюбленців.

Користувачі можуть:

* Переглядати тварин, які шукають новий дім.
* Дізнаватися інформацію про кожного улюбленця.
* Переглядати фотографії та історії тварин.
* Надсилати заявку на усиновлення.

Основна мета платформи — зробити процес пошуку нового друга простим, зрозумілим
та зручним.

# 🚀 Встановлення та запуск

Перед початком переконайтесь, що встановлено **Node.js (версія 14 або новіша)**.

## 1. Клонування репозиторію

```
git clone https://github.com/your-username/pets-founder.git
```

## 2. Перехід у папку проєкту

```
cd pets-founder
```

## 3. Встановлення залежностей

```
npm install
```

## 4. Запуск у режимі розробки

```
npm run dev
```

## 5. Відкрити сайт

Після запуску відкрийте адресу, яка з’явиться у терміналі
(зазвичай)

```
http://localhost:5173
```
